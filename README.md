# qbank-local-mvp

Local-first, offline, UWorld-inspired practice platform for university courses.

## Current scope

This repo is for the **local macOS MVP only**.

The MVP is:
- single-user
- fully offline
- no login
- local database only
- rich question authoring
- free practice
- timed blocks
- review incorrect
- spaced repetition for flashcards and short-answer
- adaptive review tracking for MCQs

## Primary goal

Build the core study loop:

**create content → build session → answer/reveal → review explanation → track review state**

## Core sections

- Library
- Authoring
- Practice Setup
- Practice Session
- Review History
- Stats Dashboard

## Do not add yet

- web platform features
- accounts/auth
- cloud sync
- moderation/reviewer systems
- public contributions
- AI generation
- heavy analytics
- import/export complexity beyond what is needed for local MVP

## Guiding principle

Make the core study loop feel excellent before adding extra features.
