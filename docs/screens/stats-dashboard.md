# Screen Spec — Stats Dashboard

## Purpose

Provide a lightweight progress overview for the local MVP.

## Day-one requirement
- streak display

## Optional lightweight supporting data
- recent sessions
- due review count
- total answered
- weak topic hints

## Layout

### Top
Primary streak card

### Secondary region
One or two lightweight summary cards if implemented

## Visual priority
1. streak
2. immediate next-action hints
3. lightweight context

## UX rules
- keep it minimal
- do not turn this into a fake enterprise analytics page
- this screen supports the study workflow; it is not the product center of gravity
